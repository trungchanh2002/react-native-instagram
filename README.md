## Getting Started

Follow these steps to run the project locally:

### Prerequisites
- Node.js
- React Native CLI

### Installation
1. Clone this repository.
2. Install dependencies using `npm install` or `yarn install`.
3. Run the project using `react-native run-android` or `react-native run-ios`.
